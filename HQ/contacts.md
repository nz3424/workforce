## How to Use This File

Each contact has a **tone profile** and any relevant **context** agents should
know. Before drafting an email, find the recipient here. If they're not listed,
default to the "Professional & direct" tone from CLAUDE.md.

---

## Template — Add new contacts in this format

```
### [Full Name]
- **Relationship:** [e.g. Manager, Professor, Friend, Recruiter]
- **Email:** [address]
- **Tone:** [Formal / Professional & warm / Casual / etc.]
- **Sign-off:** [Best / Thanks / Cheers / etc.]
- **Notes:** [Anything useful — their communication style, what they care about,
  ongoing threads]
```

## Groups / Categories

### Recruiters & Professional Contacts
- **Tone:** Enthusiastic, polished, concise
- **Sign-off:** "Best, Nick"
- **Rules:** Lead with the value prop; no long intros; always include a clear
  ask or next step

### Peers / Friends from College
- **Tone:** Casual and warm
- **Sign-off:** "Thanks" or just "Nick"
- **Rules:** Can be conversational; contractions are fine; humor is welcome

### New Colleagues (first 90 days at job)
- **Tone:** Friendly and professional; approachable but not too casual yet
- **Sign-off:** "Thanks, Nick"
- **Rules:** Ask questions freely; show enthusiasm; avoid over-promising
